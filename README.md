Neste projeto foi implementado um sistema de validação de conta, no qual o usuário deve cadastrar um e-mail e uma senha. Após o cadastro, o usuário poderá realizar o login utilizando os dados cadastrados.

Funcionalidades:
- Validação de e-mail (como @hotmail.com, @gmail.com, @yahoo.com e outros);
- Validação de senha contendo:
  - mínimo de 10 caracteres;
  - pelo menos uma letra maiúscula;
  - pelo menos uma letra minúscula;
  - pelo menos um caractere especial;
- Sistema de login com verificação de e-mail e senha;
- Limite de até 3 tentativas de login incorretas;
- Encerramento do programa após 3 tentativas inválidas ou após login realizado com sucesso.
